# Riwayat Scan (Repository)

Seluruh riwayat pemeriksaan kargo disimpan dalam modul **Scan History**.

## Melihat Riwayat
1. Buka menu **Scan History**.
2. Daftar hasil scan terakhir akan ditampilkan dengan status **Finished XRay** (Hijau) atau **Pending XRay** (Merah).
3. Anda dapat mencari data berdasarkan MAWB atau HAWB menggunakan kotak pencarian.

## Detail Snapshot
1. Klik ikon **Mata (Eye)** pada baris data untuk melihat snapshot lengkap.
2. Jendela modal akan menampilkan:
    * Citra Top View X-Ray.
    * Citra Side View X-Ray.
    * Foto Fisik Barang (jika tersedia).
3. Klik pada gambar untuk memperbesar tampilan (Zoom).

## Pengulangan Scan (Redo)
Jika terjadi kesalahan pada proses scan, Anda dapat menekan ikon **Rotate** untuk mengulangi proses screening item tersebut. Data lama akan tetap tersimpan di riwayat sebagai referensi.
