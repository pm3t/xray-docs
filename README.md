# X-Ray Offline System

Selamat datang di panduan pengguna **X-Ray Offline System**.

Sistem ini dirancang untuk memproses pemeriksaan kargo (screening) secara efisien dalam lingkungan offline. Sistem ini mendukung integrasi dengan perangkat video capture card untuk mengambil citra X-Ray secara real-time dan mencatat riwayat pemeriksaan secara lokal.

## Fitur Utama
* **Manajemen Manifest:** Pengelolaan data kargo melalui input manual atau impor massal via Excel.
* **Screening Real-time:** Pengambilan gambar dari kamera Top View dan Side View.
* **Local Repository:** Penyimpanan data pemeriksaan secara aman di workstation lokal menggunakan IndexedDB.
* **Offline First:** Seluruh operasional inti dapat berjalan tanpa koneksi internet.

## Persyaratan Sistem
* Browser Google Chrome (Versi terbaru direkomendasikan).
* Koneksi USB untuk capture card kamera X-Ray.
* Kapasitas penyimpanan lokal yang cukup untuk database citra.
